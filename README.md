# Algoritmos e Lógica de Programação
<small>*Feito por Vitor Oliveira*</small>

Repositório dedicado ao estudo de algoritmos e lógica, contendo resoluções de problemas em diferentes linguagens de programação.

## 📚 Sobre o Curso
Os exercícios fazem parte do curso **"Algoritmos e Lógica de Programação"** do **Prof. Dr. Nélio Alves**, cobrindo desde fundamentos básicos até estruturas de dados.

## 🛠️ Tecnologias e Ferramentas
O curso aborda as seguintes linguagens, que serão organizadas conforme o progresso:
* **VisualG** (Lógica inicial)
* **C**
* **C#**
* **Java**
* **Python**

## 📂 Organização do Repositório
Cada tecnologia possui sua própria pasta raiz, dividida por capítulos:
* `visualg/`: Exercícios desenvolvidos em Portugol.
* `c/`: (Em breve) Exercícios desenvolvidos em C.
